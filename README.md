# IoT Smart Waste Management System
### Embedded Systems Project | B.S. Artificial Intelligence

A functional IoT prototype designed to automate waste disposal and optimize collection cycles through real-time depth monitoring and cloud-based alerts.

## 🚀 Concept & Functionality
The project addresses the efficiency of waste management by automating the physical interaction and providing data-driven insights into bin capacity.
* **Automated Lid:** Utilized proximity sensors to trigger a servo-motor, enabling touchless disposal.
* **Capacity Monitoring:** Implemented ultrasonic sensors to measure the "Fill Level" of the bin in real-time.
* **Cloud Connectivity:** Integrated a mobile application to display live status and send push notifications when the bin reached maximum capacity.

## 🛠️ Tech Stack
* **Microcontroller:** Arduino / ESP8266 (WiFi-enabled).
* **Sensors:** HC-SR04 Ultrasonic Sensor, PIR/Infrared Proximity Sensor.
* **Connectivity:** WiFi / MQTT Protocol for mobile app integration.
* **Actuators:** Servo Motor for lid mechanism.

## 📋 Evolution of Design
Initially envisioned as an AI-based sorting system, the project evolved into a robust **Digital Logic Design (DLD)** and **IoT** application. This pivot focused on perfecting the reliability of sensor-actuator loops and ensuring seamless data transmission to the end-user.
